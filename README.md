# Flutter Portfolio Site (Web Build)

This is the **web build** of the Flutter Links Site project.  
Check out the [main branch](https://github.com/dileepadev/flutter-portfolio-site) for the full source code and project details.

## Deployment

The web build is deployed to **GitHub Pages**.  
You can access the live version here: [https://dileepadev.github.io/flutter-portfolio-site](https://dileepadev.github.io/flutter-portfolio-site)

## About

This branch contains the compiled web version of the project generated from Flutter.  
It includes all the HTML, CSS, JS, and assets required to run the site in a browser.

Use this build if you want to **host the site directly** without building from source.
